# Hierarchical-Actor-Critic-M2SVCC

This is an ongoing project on the implementation of the Hierarchical Actor Critic (HAC) algorithm described in the paper, [Learning Multi-Level Hierarchies with Hindsight](https://arxiv.org/abs/1712.00948) (ICLR 2019), in PyTorch to Multimodal, MultiScale Signal-Vehicle Coupled Control.


## References

- Official [Paper](https://arxiv.org/abs/1712.00948) and [Code (TensorFlow)](https://github.com/andrew-j-levy/Hierarchical-Actor-Critc-HAC-)
- Code: [HAC git repository for PyTorch](https://github.com/nikhilbarhate99/Hierarchical-Actor-Critic-HAC-PyTorch)
